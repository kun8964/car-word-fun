# 车车大联盟 Creative Lab

面向 5-15 岁儿童的汽车主题创意学习娱乐网站原型。

当前版本是单页静态体验页，不包含多用户体系、账号系统或二级页面。

## 本地运行

这是一个纯静态项目，直接打开 `index.html` 即可，也可以用本地静态服务运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173/
```

## 页面特性

- 100vh 整屏滚动与吸附切屏
- 视差滚动和进入/离开动画
- 自动播放、静音、循环的视频主视觉
- 儿童科技课程风格的轻量导航、圆角按钮和商业化展示屏
- 所有导航均为当前页面内锚点跳转
