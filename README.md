# 车车大联盟单页游戏中心

面向 5-15 岁儿童的汽车主题创意学习娱乐网站原型。

当前版本是单页静态游戏中心，不包含多用户体系、账号系统、预约流程或二级页面。

## 本地运行

这是一个纯静态项目，直接打开 `index.html` 即可，也可以用本地静态服务运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173/
```

## 页面特性

- 100vh 整屏滚动与吸附切屏
- 视差滚动和进入/离开动画
- 自动播放、静音、循环的视频主视觉
- 儿童科技游戏风格的轻量导航、圆角按钮和商业化展示屏
- 所有导航均为当前页面内锚点跳转

## 游戏模式方向

当前页面内的游戏模式卡片可以直接点击切换，会在同一页更新当前玩法说明。

- 车车找词：看车模画面，选择正确的中文或英文词卡。
- 零件配对：把轮胎、车灯、车门等部件和名称配起来。
- 路线排序：按启动、转弯、加速、停车，把动作顺序排对。
- 小小讲解员：用给定词卡讲一句作品说明，练观察和表达。
